📊 Inferential Statistics - Notes

1. Definition

Inferential statistics is used to draw conclusions about a population using sample data.

Example: Survey 1000 voters to predict election results.

2. Why We Need It

  • Population data is large or unavailable
  • Helps in predictions and decision-making

Example: Testing app feature on 200 users instead of all users.

3. Techniques

(A) Confidence Interval

CI = x̄ ± Z(α/2) × (σ / √n)

Gives a range where the true value lies.

Example: Height = 170 → CI = (168, 172)

(B) Hypothesis Testing

  • H₀: No effect
  • H₁: Has effect
  • If p-value < 0.05 → Reject H₀

Example: p = 0.03 → New algorithm is better

(C) Central Limit Theorem

X̄ ~ N(μ, σ/√n)

Sample mean becomes normal for large samples.

4. Errors

  • Type I Error: Reject true H₀ (False Positive)
  • Type II Error: Accept false H₀ (False Negative)

Power: 1 − β

5. Tests

Parametric

  • Z-test
  • T-test
  • ANOVA

Non-Parametric

  • Chi-Square
  • Mann-Whitney

6. Real-Life Example

Check if new delivery algorithm is faster:

  • H₀: No improvement
  • H₁: Faster
  • If p-value < 0.05 → Better algorithm

Example: Saves 2–5 minutes delivery time

🎯 Summary

Use sample data to make population-level decisions with probability.
